Абсолютно твердое тело – такие тела, деформациями в которых под действием внешних сил можно пренебречь.

Вращательное движение МТ – движение точки по окружности.

Вращательное движение АТТ – движение АТТ, при котором все его точки движутся по окружностям, центры которых лежат на одной прямой, которая называется осью вращения.

Угол поворота – вектор, модуль которого равен углу, на который повернулось тело, а направление определяется с помощью правила правого винта.
Правило правого винта:
Если правый винт вращать по направлению вращения тела, то его поступательное движение покажет направление угла поворота.

Радиан – угол, соответствующий дуге, длина которой равна её радиусу.

Мгновенная угловая скорость – векторная физическая величина, равная пределу отношения угла поворота, к промежутку времени, за который это изменение произошло, при условии, что этот промежуток времени стремится к нулю.

Мгновенное угловое ускорение – векторная физическая величина, равная пределу отношения изменения угловой скорости к промежутку времени, за который это изменение произошло, при условии, что этот промежуток стремится к нулю.

Равноускоренное движение по окружности – движение, при котором угловое ускорение - постоянная во времени величина, и угловая скорость увеличивается.

Равномерное движение по окружности - движение по окружности с постоянной по модулю скоростью.

Период вращения – это физическая величина, равная отношению времени к количеству оборотов, совершённых за это время.
T = t / N
[T] = [c]
Физический смысл: Период вращения показывает время одного полного оборота тела на угол 2π радиан.

Частота вращения – это физическая величина, равная отношению количества оборотов ко времени, за которое эти обороты были совершены.
v = N / t
[v] = [c^-1]
Физический смысл: Частота вращения показывает число оборотов в единицу времени.

Формула связь периода и частоты вращения:
T = 1 / v
v = 1 / T
Связь между угловой скоростью и частотой вращения:
ω = 2π * v
Связь между угловой скоростью и периодом вращения:
ω = ∆ф / ∆t = 2π / T
Связь между угловой скоростью и линейной скоростью:
v = ωR (в векторной форме)
Пояснение:
Векторное произведение угловой скорости тела на радиус-вектор точки относительно оси вращения.
Модуль нормального ускорения:
a n = v^2/R
Связь между нормальным ускорением, угловой скоростью и линейной скоростью (формула через векторное произведение):
a n = ω*v (в векторной форме)

Нормальное ускорение - это векторная физическая величина, равная пределу отношения изменения нормальной составляющей скорости, за которое это изменение произошло, при условии, что этот промежуток времени стремится к нулю.
a n = lim(∆t→0) (∆v_n)/∆t
[a n]=[м/с^2]
Физический смысл: характеризует быстроту изменения скорости по направлению.

Тангенциальное ускорение - это векторная физическая величина, равная пределу отношения изменения тангенциальной составляющей скорости, за которое это изменение произошло, при условии, что этот промежуток времени стремится к нулю.
a т = lim(∆t→0) (∆v_т)/∆t
[a т ]=[м/с^2 ]
Физический смысл: характеризует быстроту изменения модуля вектора скорости.
